Como realizar una entrega

Finca

Selecciona en el desplegable la finca de la que van a salir los productos

Seleccionar el producto

Después de un par de segundos, si pulsas en la flecha del cuadro de productos puedes ver todos los productos a entregar

Introducir las cantidad

Escribe el numero de etiquetas que quieres sacar de ese producto

Pulsa + para añadir las etiquetas

Una vez tengas todas las etiquetas añadidas pulsa "IMPRIMIR"
